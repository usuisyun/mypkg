# mypkg
ロボットシステム学　課題2

# １．概要  
講義内で作成したプログラムです。改造点はありません。

# ２．環境  
Raspberry Pi4(ubuntu20.04 LTS)

# ３．実装内容   
0から1ずつ加算した数値を出力

# ４．インストール・実行手順  
`$ catkin_ws/src/mypkg`上に  
`https://github.com/usuisyun/mypkg.git`  
`roscore`  
`chmod +x count.py`  
`rosrun mypkg count.py`  
`rostopic echo /count_up`
